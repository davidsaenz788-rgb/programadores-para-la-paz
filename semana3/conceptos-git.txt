Repositorio:
Un repositorio es un espacio donde se almacenan los archivos y el historial de cambios de un proyecto.
Permite organizar el trabajo y mantener control sobre las diferentes versiones del software.

Commit:
Un commit es un registro de los cambios realizados en un proyecto.
Cada commit guarda una versión específica del trabajo con un mensaje descriptivo.

Push:
El comando push permite enviar los cambios del repositorio local hacia GitHub u otro repositorio remoto.
Esto ayuda a respaldar el proyecto y compartir el trabajo con otras personas.
